# Product2Video MVP

基于 PRD 实现的第一版 MVP：
- 首页（价值说明 + CTA）
- 生成页（上传 3-8 张图 + 输入卖点 + 选择模板 + 生成）
- 结果页（视频预览 + 模板 + 卖点 + 下载）
- Mock API（约 2.2 秒假生成流程）

## Run

```bash
npm install
npm run dev
```

打开 http://localhost:3000

## 说明

- 当前视频为 `public/demo-video.mp4`（可替换为你的 demo 文件）
- 无数据库、无登录、无支付，符合 MVP 第一阶段范围
